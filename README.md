ASCII String & Number Pattern Generator (Java)

This project is a Java-based console application that prints **large ASCII art patterns** for:
- Individual characters (A–Z)
- Full strings
- Ranges of characters (A–Z)
- Numbers (0–9)

It also allows you to find the **alphabetical position** or **numeric index** of characters and digits.

Features

1. Character Position
Shows the alphabetical order of each character (A → 1, B → 2, …).

2. Single Character Pattern
Prints a big ASCII representation of any let

3. String Pattern
Converts a full string into large ASCII art.

4. Range of Characters
Prints large ASCII letters between two characters  
(e.g., A → E prints A, B, C, D, E patterns).

5. Number Position
Prints numeric positions for digits (0–9).

6. Number Pattern
Prints large ASCII patterns for numbers.
